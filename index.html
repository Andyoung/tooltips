bubbles.css

纯css打造的工具提示

Github Project Download CSS

示例

气泡方向：上、下、左、右

<button data-bubbles="向上提示" data-bubbles-pos="up">向上提示</button>
<button data-bubbles="向左提示" data-bubbles-pos="left">向左提示</button>
<button data-bubbles="向右提示" data-bubbles-pos="right">向右提示</button>
<button data-bubbles="向下提示" data-bubbles-pos="down">向下提示</button>

气泡长度：小、中、大、宽度适合于元素

<button data-bubbles-length="small" data-bubbles="我是小提示" data-bubbles-pos="up">我是小提示</button>

<button data-bubbles-length="medium" data-bubbles="我是中等提示" data-bubbles-pos="up">我是中等提示</button>

<button data-bubbles-length="large" data-bubbles="我是大提示" data-bubbles-pos="up">我是大提示</button>
<button data-bubbles-length="xlarge" data-bubbles="我是一个超大的提示" data-bubbles-pos="up">我是一个超大的提示</button>
<button data-bubbles-length="fit" data-bubbles="宽度适合于元素提示" data-balloon-pos="up">宽度适合于元素提示</button>

使用方法

在HTML上添加bubbles.min.css

<link rel="stylesheet" href="bubbles.min.css">

给标签添加 data-bubbles 和 data-bubbles-pos 属性

<span data-bubbles="我是提示" data-bubbles-pos="up">bubbles</span>
